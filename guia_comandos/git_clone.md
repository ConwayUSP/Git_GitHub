# `git clone`

Voltar ao [Guia de Comandos](README.md) 

## O que faz?
O comando `git clone` tem a função de criar uma cópia exata (um clone) de um repositório Git existente remotamente (na internet) para a sua máquina local. Ele baixa todos os arquivos, branches e todo o histórico do projeto de uma só vez.

## Formas de usar

`git clone <url-do-repositorio>`: clona o repositório na URL especificada criando uma nova pasta com o nome do projeto;
`git clone <url-do-repositorio> <nome-da-pasta>`: clona o repositório dentro de uma pasta com o nome que você definiu.

## Formas de usar (pelo GitHub Desktop)

Pelo programa, vá em File > Clone repository... (ou Ctrl + Shift + O). Você verá uma lista dos seus repositórios. Escolha um ou cole a URL na aba "URL", defina o caminho local onde quer salvar e clique em "Clone".

## Formas de desfazer

Para "desfazer" um clone, basta abrir o explorador de arquivos do seu computador e deletar a pasta do projeto que foi baixada.

## Palavras chaves

- `cópia`, `baixar projeto`, `url`, `repositório inteiro`

# Desafio Final

## Objetivo

Como provas são chatas, o objetivo desse desafio é colocar seus conhecimentos de Git(Hub) à prova através de um projeto prático que simula um cenário real de desenvolvimento. Para isso criamos um repositório exclusivo para esse desafio: o [To Commit or Not to Commit](https://github.com/ConwayUSP/To-Commit-or-Not-to-Commit)! Nele, você terá algumas missões para cumprir:

- [ ] Faça um fork do repositório mencionado (copiando todas as branchs) para a sua conta pessoal do GitHub
- [ ] Clone o repositório para a sua máquina local para que seja possível trabalhar nele
- [ ] Existe um commit que foi introduzido "por engano" no histórico, e ele precisa ser removido. Encontre o intruso e o remova da maneira que achar mais adequada (observe o histórico atentamente que você irá encontrá-lo com certeza)
- [ ] Alguns atos estão fora das suas devidas pastas. Mova-os para o lugar certo, da maneira adequada.
- [ ] Um capítulo de Hamlet não havia sido incluído, então criou-se uma branch `fix/add-act4-scene` para resolver esse problema. Faça o merge dessa branch com a `main` e resolva o possível conflito que irá surgir (analise atentamente para identificar qual versão do arquivo deve ser mantida)
- [ ] O repositório está um pouco vazio somente com Hamlet, você não acha? Para resolver isso, crie uma nova branch dedicada a uma outra obra de Sheakespare (todas elas podem ser encontradas [aqui](https://shakespeare.mit.edu/)) e adicione algumas cenas (não precisa ser a obra inteira, só um gostinho mesmo haha). Depois, faça o merge dessa branch com a `main` também. Lembre-se de manter um padrão na estrutura e mensagem dos commits.
- [ ] Por fim, crie um Pull Request para o repositório original (da `main` do seu fork para a `main` do repositório original) explicando as mudanças que foram feitas de forma geral.

Para comunicar que o desafio foi concluído, entre em contato com algum membro de OP responsável pelo projeto e envie o link do seu PR. Assim, poderemos analisar o seu trabalho e, se tudo estiver certo, você ganhará seu selo muito em breve!

# Git & GitHub: Guia de Aprendizado e Consulta

Olá, meu caro amiguinho! Esse é o inicio do material didático da Conway que irá abordar duas ferramentas bem bacanas, essenciais para o desenvolvimento de softwares e de projetos no geral: Git e GitHub.

Aqui, você poderá aprender do zero como usar essas duas ferramentas ou simplesmente utilizar esse repositório como material de consulta.

## O que é Git e GitHub?

Git é um sistema de controle de versão distribuído que permite que você acompanhe as mudanças no seu código ao longo do tempo. Ele facilita a colaboração entre desenvolvedores, permitindo que múltiplas pessoas trabalhem no mesmo projeto simultaneamente sem conflitos.

GitHub, por outro lado, é uma plataforma baseada na web que hospeda repositórios Git. Ele oferece uma interface amigável para gerenciar projetos, colaborar com outros desenvolvedores e compartilhar seu código com o mundo.

De forma sucinta, essas duas ferramentas são fundamentais para o desenvolvimento não só de softwares, mas de qualquer projeto que envolva arquivos e mudanças constantes. Assim, se você deseja trabalhar em equipe, compartilhar seu trabalho ou simplesmente manter um histórico organizado das suas mudanças, aprender Git e GitHub é essencial.

## Objetivo 

O objetivo deste material é fornecer uma base sólida para que você possa entender os conceitos fundamentais do Git e GitHub, aprender os comandos mais importantes e aplicar esse conhecimento em situações práticas. Além disso, depois de já ter consolidado sua base, você poderá usar esse material como um guia de consulta rápida para resolver dúvidas ou relembrar comandos específicos. Incrível, não é mesmo!?

## Ferramentas Utilizadas

- **Git**: para controle de versão e gerenciamento de código-fonte.
- **GitHub**: para hospedagem de repositórios e colaboração online.
- **Visual Studio Code**: para edição de código e integração com Git.
- **GitHub Desktop**: para uma interface gráfica de gerenciamento de repositórios Git.
- **Terminal**: para execução de comandos Git e navegação no sistema de arquivos.

Todas essas ferramentas são gratuitas e serão utilizadas ao longo do material para demonstrar os conceitos e práticas do Git e GitHub. Se você ainda não as tem instaladas (com exceção do terminal, que é nativo do sistema operacional), não se preocupe! Assim que iniciarmos o guia prático, vamos orientá-lo passo a passo sobre como configurar tudo isso para começar a usar o Git e GitHub sem complicações.

## Estrutura do Material

O material está dividido em três seções principais:

1. **Guia de Comandos**: Uma lista detalhada dos comandos mais importantes do Git, com explicações e exemplos de uso.
2. **Guia Prático**: Tutoriais passo a passo que mostram como realizar tarefas comuns com Git e GitHub.
3. **Glossário de Conceitos**: Definições e explicações dos principais termos relacionados ao Git e GitHub.

Cada seção é projetada para ser fácil de navegar, permitindo que você encontre rapidamente as informações que precisa.

Sinta-se à vontade para explorar o material e aprender no seu próprio ritmo. Se tiver alguma dúvida ou sugestão, não hesite em entrar em contato!
